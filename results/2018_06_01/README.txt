Combined_GWAS_PheWAS.csv contains the SNPs from the GWAS that were found in the PheWAS with all the PheWAS data.

missing_snps.csv contains the GWAS SNPs that were not found in the PheWAS.

repetitions.csv contains the number of times each SNP was found in the PheWAS.

set_output.csv is a double check of which SNPs are found in both the GWAS and PheWAS using Python sets.

snps folder contains csv files for each snp sorted by p-value, then odds, then phewas_string
